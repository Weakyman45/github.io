Leetcode Problem 530: Minimum Absolute Difference in BST

Problem Description

Given a binary search tree (BST), we aim to find the minimum difference between the values of any two nodes.

Approach and Thought Process

The problem can be efficiently solved using an in-order traversal of the BST. An in-order traversal of a BST visits nodes in ascending order, which makes it ideal for comparing adjacent nodes to compute the minimum difference. The steps are as follows:

  1. Perform an in-order traversal of the BST and store the values in a sorted list.
  2. Iterate through the list of values and calculate the difference between adjacent values.
  3. Track the minimum difference encountered during this iteration.
  4. Return the minimum difference as the final result.

Solution Code

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    vector inorder;
    void inorderRecursive(TreeNode* node) {
        if (node != nullptr) {
            inorderRecursive(node->left);
            inorder.push_back(node->val);
            inorderRecursive(node->right);
        }
    }
    int getMinimumDifference(TreeNode* root) {
        inorderRecursive(root);
        int minDiff = INT_MAX;
        for (int i = 1; i < inorder.size(); i++) {
            minDiff = min(minDiff, inorder[i] - inorder[i - 1]);
        }
        return minDiff;
    }
};
